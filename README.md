Task Planner (in progress)

This is a backend project for a task planner application. 
The goal of the project is to help users create tasks, track their progress, and manage their daily activities.

Technologies:
- Python
- Django
- PostgreSQL

Current features:
- project structure
- database models
- basic task management logic

Planned features:
- user registration and authentication
- task creation and editing
- progress tracking
- REST API for frontend integration

Status:
Project is currently in development.
